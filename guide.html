Quick start

Less thinking, more plotting
1Pick a Stage preset or type a custom Width and Depth.
2Add items from the Item Library (guitar, vox, wedges, power).
3Tap an item to select it, then edit label, input, mix, notes, size, rotation.
4Use Print for a clean one page handoff, or Export Project JSON to share.
Autosave is on. Your plot sticks around between sessions unless you reset or clear it.

Stage controls

Layout vs view, they are different on purpose
Stage preset and dimensions
  • Width is left to right.
  • Depth is upstage to audience.
  • The stage always fits inside the square at 100%.
Tip: If you are not sure, start with Medium and adjust later.
View tools
  • View Zoom zooms your view inside the square (detail mode).
  • Reset View snaps back to 100%.
  • Flip switches Stage View and Audience View (for sanity checks).
  • Units toggles ft and m for display (internally stored as feet).
Reset vs Clear
  • Clear clears selection only (it does not delete items).
  • Reset Stage is the big red button behavior: it wipes the plot back to a clean stage.
If you hit Reset Stage by accident, you are allowed one dramatic sigh. After that, add more wedges.

Selection and editing

Single select, multi select, and mild wizardry
Select items
  • Tap an item to select it.
  • Shift click (or Cmd or Ctrl click) to multi select.
  • Escape clears selection.
  • Delete removes selected items.
Multi select is perfect for dragging a whole drum world, or making all wedges the same color, or both.
Edit selected item in the Info panel
  • Label, Input, Mix, Notes
  • Item Width, Item Depth
  • Scale slider (footprint)
  • Rotation slider, plus Rotation reset
  • Color picker (applies to the selected item, or the whole multi selection in v1.6)
Color is purely for readability. It does not summon more power drops, sadly.
Under stage tools (Selected item tools)
  • Nudge buttons move items in small steps.
  • Shift + Arrow keys nudges selected items too.
  • Width and Depth fields let you type exact footprints quickly.
  • Scale slider changes footprint scale.
  • Pinch to scale on touch screens: two fingers on the selected item and pinch.
Remember: Width and Depth set the base footprint. Scale multiplies it. View Zoom only changes your view.

Keyboard shortcuts

For the keyboard heroes
These work when you are not typing in a text field.
Shift + Arrow keys nudge selected items
Delete or Backspace delete selected items
Cmd or Ctrl + D duplicate selected items
Esc clear selection

Export, import, print

Handoff mode
Export Project JSON
  • Creates a shareable file containing your stage, items, and notes.
  • Use it to send to bandmates or keep versions for different venues.
Import Project JSON
  • Loads a saved project back into the app.
  • Older versions are migrated forward automatically when possible.
Print
  • Print makes a clean one page sheet.
  • The app temporarily snaps View Zoom to 100% for printing so it fits nicely.
  • Tip: Fill Band / Show name and Show notes so the sheet is actually useful to humans.

Troubleshooting

Fixes without throwing your phone
  • My stage looks zoomed or weird - hit Reset View (100%).
  • I cannot select an item - you might be in a scrolled panel. Try tapping directly on the box.
  • My item is off the stage - change size or scale down, then nudge it back. Items are clamped to stay inside the stage.
  • Pinch does nothing - pinch works on the selected item only. Tap the item first, then pinch on it.
  • I lost my plot - import your last exported JSON. If you did not export, the app only has what is in local storage.
If all else fails, export a copy, reset stage, then import. It is the digital version of turning the amp off and on.
