Quick start

2 minute survival mode
1 Add stuff
  • Use Item Library to drop musicians and gear onto the stage.
  • Click an item to select it. Shift, Cmd, or Ctrl click to multi select.
2 Set stage size
  • Pick a preset or set custom Width and Depth.
  • Use the Toggle ft/m button if your venue speaks in meters and your band speaks in vibes.
3 Fill the one page info
  • Add Band / Show name and Show notes for the top of the printout.
  • Select an item and set Label, Input, Mix, Notes.
  • The app auto builds Inputs and Mix lists from your items.
4 Print
  • Hit Print for a clean one page handoff.
  • Print temporarily forces View Zoom back to 100 percent so your diagram is not accidentally a microscopic jazz trio.

Autosave friendly means your work is saved in your browser localStorage. Your plot lives on your device like a loyal stagehand, but only until you clear site data.

Selection, movement, and scaling

draggy, nudgy, pinchy
Select
  • Click an item to select it.
  • Shift, Cmd, or Ctrl click adds or removes items from the selection.
  • The main selected item is the one that drives the edit fields.
Move
  • Drag to move selected items.
  • Hold Shift + Arrow keys to nudge by small steps.
  • Use the under stage Nudge pad for tap nudging.
Size controls
  • Each item has base Width and Depth.
  • Scale multiplies the footprint (50 percent to 200 percent).
  • You can change size using number fields and sliders.
Touch gesture scaling
  • On touch screens, select an item and do a two finger pinch on that item to change Scale.
  • If pinch feels fussy, make sure your fingers are actually on the item, not on the stage background.

View Zoom changes your view only. It does not change item sizes or stage dimensions, it just makes your eyeballs zoom like a camera.

Color tools

v1.6 rainbow
What color does
  • Item color is a visual highlight for roles, inputs, zones, or anything your brain needs to decode quickly.
  • Color applies to the selected item, and when you have multiple items selected it can apply to all of them at once.
Recommended use
  • Blue for vocals, yellow for instruments, red for power, green for wedges.
  • Or go full chaos and color code by who is most likely to ask for “just a little more me” in the monitors.

If you do not see the color picker, it should live in the selected item tools area and or the One Page Info selected item section depending on your build.

Stage and view controls

layout vs view
Stage size
  • Use presets or set custom Width and Depth.
  • Stage always fits the square at 100 percent. That is by design so you never “lose” the stage off screen.
Flip stage
  • Stage View shows Upstage at the top.
  • Flip to Audience View when sharing with FOH, venues, or anyone who stands in front of the stage and judges your pedalboard choices.
View Zoom
  • Use View Zoom slider to zoom into the stage square.
  • Use Reset View to return to 100 percent.
Reset stage
  • Clear clears selection only, not the stage.
  • Reset Stage should wipe items and return stage settings to defaults (if enabled in your build).

If you only see Clear and it does nothing, that button is for selection, not a full reset. The reset button should be a separate action so nobody nukes a plot mid rehearsal.

Export, import, and sharing

save your masterpiece
Export Project JSON
  • Saves a portable project file you can email, text, or stash in a folder named “final FINAL actually final”.
  • Useful for bandmates, recurring gigs, and not trusting your browser cache.
Import Project JSON
  • Loads a previously exported file back into the app.
  • Import is migration friendly so older projects can still load.

  • Create a Share button that generates a URL containing a compressed project payload or a project id.

  • Optional: a “view only” mode so nobody accidentally turns the drummer into a power drop.

Keyboard shortcuts

desktop power
  • Shift + Arrow keys nudges selected items.
  • Delete or Backspace deletes selected items.
  • Esc clears selection.
  • Cmd or Ctrl + D duplicates selected items.

If your keyboard shortcuts do nothing, click outside of input fields first. Typing into notes while trying to nudge is a classic.

FAQ and gotchas

things that feel like bugs
  • Why does Clear not erase the stage Because Clear is meant to clear selection only. Stage reset should be separate so nobody rage clicks and deletes a whole show.
  • Why do my items clamp at the edges Items are kept inside the stage boundaries including their scaled footprint.
  • Why do my printed items look different Print forces View Zoom to 100 percent temporarily so your page is consistent.
  • Why are there two size control areas One is in Info, one is under the stage for fast adjustments while placing items.
  • Can I color multiple items Yes, select multiple items and apply a color once. It is the adult version of a highlighter.
